# Unreal_Plugin_Inventory



공간형 인벤토리를 만들고 플러그인 형태로 만들어서 다른 프로젝트에서도 사용가능하도록 제작되었다.
캐릭터의 Controller에 Inventory Component를 부착해서 사용할수 있도록 만들었다.
아이템을 필요한 조각들을 합성해서 만들수 있도록 구성해, 필요한 정보들을 넣을수 있게 만들었다.
이러한 아이템 구성또한 Component형태를 사용해서 단순히 Actor에 부착해서 인벤토리에 넣을수 있도록 하였다.
